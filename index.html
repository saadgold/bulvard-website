About Bulvard

We build tools we'd want to use ourselves.

Bulvard is an independent software studio focused on desktop applications that put users first. We believe software should be fast, focused, and respectful of the people who use it.

That means processing data locally whenever possible, being transparent about what our apps do, and shipping tools that solve real problems without the bloat.
